# amazon

Simplified online retail system with terminal user interface and no photos.

To run:

`make amazon`

`./amazon database_file`

Sample `database.txt` provided

## Features

* AND search
* OR search
* Adding to cart
* Viewing cart
* Buying cart
* Output updated database
